Storm SEO Pages (Internal)

These pages are generated per storm/city for outreach + demos. Create a new one by running the generator script (see below).

Generate a storm page

Run:
node scripts/make-storm-page.mjs --city "Dallas" --state "TX" --date "2026-03-03" --event "Severe Wind" --zips "75201,75202"

Then commit + push to deploy.
